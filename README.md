# 📝 Task Management System 

A full-stack Task Management System that allows admins to manage users and tasks, while users can track and update their assigned work. Built with modern technologies and a clean UI.

---

## 🚀 Features

### 👨‍💼 Admin Dashboard
- View total users, tasks, and status overview
- Create, edit, and delete tasks
- Assign tasks to users
- Manage users (update name, email, role)
- Export tasks as CSV
- Filter tasks by status

### 👨‍💻 User Dashboard
- View assigned tasks
- Update task status (Pending / Completed)
- Add comments to tasks
- Filter tasks by status

---

## 🛠️ Tech Stack

### Frontend:
- React.js (Vite)
- Tailwind CSS

### Backend:
- Node.js
- Express.js

### Database:
- SQLite (lightweight, file-based database)
